# Решение команды На Хайпе

### Запуск проекта через Docker

Создать .env, переменные окружения
Нужно создать .env файл и переместить туда содержимое файла .env.dist
```
cp .env.dist .env
```

Запустить докер
```
docker-compose -f docker-compose.yml up -d
```

### Веб интерфейс
```
localhost:8000
```
